Very straightforward. Just write these lines in your VirtualHost, Location or Directory configuration.

Order deny,allow

Deny from all

AuthName "htaccess password prompt"

AuthUserFile /path/to/.htpasswd

AuthType Basic

Require valid-user

Allow from 172.17.10.1

Satisfy Any
